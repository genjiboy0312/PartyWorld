<h2>💡 Why?</h2>

<ul>
  <li><b>플랫폼 선택 이유:</b> PC와 콘솔에는 다양한 파티 게임이 존재하지만, <b>모바일 플랫폼에서는 부족</b>하다고 느껴 제작을 시작함.</li>
  <li><b>시장 인사이트:</b> 아기자기한 그래픽의 파티 게임에 대한 <b>수요는 많지만 공급이 적음</b>.</li>
  <li><b>개발 동기:</b> 평소 이러한 장르를 좋아하고, <b>Photon / Addressable</b> 등 네트워크 및 리소스 관리 기술을 학습 및 실습하기 위함.</li>
</ul>
